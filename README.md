# KeymastersKeepGames
Taking my shot at making a KMK implimentation, so here we are!

## Fitness Boxing feat HATSUNE MIKU
Only includes the "Miku Exercise" songs - which despite the name is all the vocaloids. So nothing from the "Basic Workout" tab. 

## Pokopia
Please give me any feedback you have, I feel like some numbers may need to be adjusted, but currently in my testing they all come out fine and not *too* grindy in my opinion! Please let me know if you feel otherwise -^ ^-
Default challenges are;
Gather X materials
Harvest X growables
Take a photo with Pokemon who have X abilities
Appraise X lost relics
Find X feathers
Create X different habitats
Find X treasure spots

Additional challenges to toggle on/off;
Complete X daily challenges
Play a round of X minigame(look that way, jump rope, etc)
Cook X meals
Craft X resources(lumber, paper, tinkagears)
